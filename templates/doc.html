/api/installations/arrondissement/{nom_arrondissement} get

get

Fournit la liste des installations qui se trouve dans l'arrondissement spécifié.

get /api/installations/arrondissement/{nom_arrondissement}

Fournit la liste des installations qui se trouve dans l'arrondissement spécifié.

  • Request
  • Response

URI Parameters

  • nom_arrondissement: required(string)

HTTP status code 200

Liste des installation dans l'arrondissement spécifié.

Body

Media type: application/json

Type: array of object

Items: Installations

  • ID: required(integer)
  • Nom: required(string)
  • Type: required(number)

Example:

­ ID=114 Nom='Parc Ladauversière' Type='Piscine extérieure', ID=118 Nom='Parc Delorme' Type='Piscine extérieure'

HTTP status code 404

Le nom de cette arrondissement n'existe pas.

/api/installations/recentes get

get

Fournit la liste des installations dont l'état a été changé en 2021.

get /api/installations/recentes

Fournit la liste des installations dont l'état a été changé en 2021.

  • Response

HTTP status code 200

La liste des installations dont l'état a été changé en 2021.

Body

Media type: application/json

Type: array of object

Items: Patinoire

  • ID: required(integer)
  • Nom: required(string)
  • Arrondissement: required(string)
  • Conditions: required(array of Condition)

    Items: Condition

    • Date_heure: required(string)
    • Ouvert: required(number)
    • Deblaye: required(number)
    • Arrose: required(number)
    • Resurface: required(number)

Example:

­ ID=114 Nom='Parc Ladauversière' Arrondissement='Ski', Ouvert=0 Deblaye=1 Arrose=1 Resurface=0

HTTP status code 404

Aucun installations a vu son état changé en 2021.
